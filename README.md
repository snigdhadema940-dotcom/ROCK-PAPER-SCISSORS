# ROCK-PAPER-SCISSORS
a interactive rock paper scissors game made using HTML,CSS and Javascript 
##FEATURES
-here the player can choose rock paper or scissors using the respective buttons
-when ever the player clicks on a button then a random computer choice is also generated
-computer choice is generated using the Math.random() function and a array of choices which contains rock,paper,scissors
-the based on the result a message is displayed
-and points of the player and computer gets updated and displayed accordingly
##TECHNOLOGIES USED
-HTML
-CSS
-Javascript
